## 💡 建议的解决方案

### 方案 1: 最小化修改（推荐）

**只修改最关键的问题，不做大规模重构**

#### 需要修改的地方：
1. **提高文本对比度** - 只修改几个关键的颜色值
2. **确保设置界面可见** - 检查是否有 CSS 隐藏了设置界面
3. **保持原有的视觉风格** - 不改变背景色和整体布局

#### 具体操作：
```css
/* 在 index.css 末尾添加少量修复 */

/* 提高亮色主题下的文本对比度 */
.text-gray-400:not(.dark *) {
  color: #6B7280;
}

.text-gray-500:not(.dark *) {
  color: #4B5563;
}

.text-gray-600:not(.dark *) {
  color: #374151;
}

/* 确保图标可见 */
button svg {
  opacity: 1;
}

/* 确保设置模态框可见 */
[class*="fixed"][class*="inset-0"] {
  z-index: 9999;
}
```
