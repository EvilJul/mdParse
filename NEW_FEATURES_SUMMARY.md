## 🚀 使用方法

### 文件标签页
1. 打开多个文件
2. 标签自动显示在顶部
3. 点击标签切换文件
4. 点击 X 关闭文件
5. 黄点表示未保存

### 自动保存
1. 打开已保存的文件（有文件路径）
2. 编辑内容
3. 等待 30 秒
4. 自动保存，右上角显示提示

### 搜索替换
1. 按 `Ctrl/Cmd + F` 打开搜索
2. 输入搜索内容
3. 按 Enter 查找下一个
4. 输入替换内容
5. 点击"替换"或"全部"
6. 按 Esc 关闭
